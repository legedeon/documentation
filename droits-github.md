# Droits Github
## Creer une clef ssh
- Ouvrir un terminal
- Creer une clef ssh : ssh-keygen puis accepter les options par defaut
- Copier la clef dans le presse papier : cat .ssh/id_rsa.pub
- Aller dans le Profil de l'utilisateur Github, rubrique SSH et ajouter la clef
